# Input Priority

Use sources in this order when they disagree:

1. sample sheet, FileMaker export, or sequencing run export
2. direct researcher email clarifications
3. GEO template values already entered by the user
4. manuscript methods and results
5. general inference from experiment design

## Why

- Sample sheets are best for exact row mapping and replicate labels.
- Researcher email clarifications often contain curator-ready wording.
- Existing workbook entries may reflect prior manual curation.
- Manuscripts are strong for assay, organism, protocols, and study summaries but may not enumerate library order.

## Mapping Guidance

When mapping GEO rows to samples:

- first compare library IDs in the workbook to any sample sheet IDs
- use replicate blocks exactly as exported
- do not reorder rows unless there is explicit evidence
- if the workbook already contains library names, preserve them and map metadata onto those rows
